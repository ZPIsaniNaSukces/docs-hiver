# Mobile App Features

Detailed documentation of mobile app features and user flows.

## Presence Check-In

### Manual Check-In
1. User opens the app
2. Taps "Check In" button
3. Location verification (optional)
4. Confirmation displayed
5. Backend records presence

### Automatic Check-In via IoT
1. User approaches office with IoT beacon
2. Device detects user's phone
3. App prompts for check-in confirmation
4. User confirms or auto-confirms (based on settings)
5. Presence recorded

## Dashboard

The main dashboard displays:
- Current presence status (checked in/out)
- Today's schedule
- Upcoming tasks
- Recent notifications
- Quick action buttons

## Presence History

View historical presence data:
- Calendar view of past check-ins
- Daily/weekly/monthly summaries
- Presence statistics
- Export functionality

## Task Management

Employees can:
- View assigned tasks
- Update task status
- Add comments
- Set task reminders
- Mark tasks as complete

## Notifications

Types of notifications:
- Task assignments
- Presence reminders
- Schedule changes
- System announcements
- Direct messages from managers

Notification settings:
- Enable/disable push notifications
- Notification sound and vibration
- Quiet hours

## Profile & Settings

**Profile Section**:
- Personal information
- Contact details
- Profile photo
- Company affiliation

**Settings**:
- Notification preferences
- IoT device management
- Auto check-in settings
- Language and region
- Privacy settings

## Offline Functionality

The app supports offline mode:
- View cached presence history
- Queue check-in/out actions
- Sync when connection restored
- Offline task viewing
