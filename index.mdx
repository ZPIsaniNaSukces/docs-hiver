# HiveR Documentation

Welcome to the HiveR system documentation. HiveR is a comprehensive workforce management platform that helps companies manage employee presence, tasks, and operations through web and mobile interfaces.

## System Overview

HiveR is a microservice-based system consisting of:

- **Backend Services**: Main service, Presence microservice, and Tasks microservice
- **Mobile Application**: Employee-facing app for presence management and IoT device interaction
- **Web Application**: Management interface for managers and administrators
- **Infrastructure**: PostgreSQL databases, Kafka message bus, and AWS SNS for notifications

## User Roles

**Employee**: Registered employee who uses the mobile app to manage in-office presence and interact with IoT devices.

**Manager**: Uses the web application to manage workforce and monitor operations.

**Admin**: Company owner with full system administration capabilities through the web interface.
